PASSWORD RESET PAGE

Welcome to the Charles River Laboratories password reset page. This page can be used to-

  • Register for Self-Service Password Reset
  • Reset your password when you have forgotten it
  • Unlock your account if it has been locked out
  • Reset your password when you have forgotten it or if it expired

All of these tasks can be performed without resorting to have to call the Service Desk.

To be able to take advantage of these features you are REQUIRED to register for this service before you need / try to use it.

If you have not registered already, then please click here

You can see below links to video tutorials on how you can both register and use the service

If you have any queries or issues with this service please contact the IT Service Desk for further assistance

1. If you are working remote and wish to reset your password, please follow these steps

2. Logon to your computer using your old password

3. Open a browser and go to this page (https://password.criver.com/) and reset your password

4. Close the browser

5. Launch the company VPN and sign-in using your new password. Wait until the VPN is connected

6. Press CTRL+ALT+DEL to lock the Windows desktop

7. Unlock the Windows desktop using your new password. This will update the cached credentials stored on your computer so they contain your new password
